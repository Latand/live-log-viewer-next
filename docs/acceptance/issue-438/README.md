# Issue 438 visual evidence

These images were captured from the repository's isolated demo runtime with a
synthetic, pipeline-only project named `pipeline-lab`.

- `pipeline-groups-desktop.png` shows one expanded group, the displaced second
  group, and the camera framing both world-space rectangles.
- `pipeline-groups-390px.png` shows the same two synthetic pipelines in the
  390 px mobile layout.

The captures are scoped to the board and mobile main surfaces. Account panels,
resource readings, filesystem paths, and conversation content are excluded.
